# ui-playground

## Commands
```bash
npm run dev     # Node ≥22.12.0
npm run build
npm run preview
```

Tailwind 4: Vite plugin, CSS-first, no config.js/PostCSS — use `@theme` CSS block.
Three.js: `<Canvas>` from `@react-three/fiber`, helpers from `@react-three/drei`.
Framer Motion: `motion.*` components, `useAnimation`/`useScroll` hooks.
Astro 6 islands: `client:load` / `client:visible`.
